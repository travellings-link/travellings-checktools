# 开往 巡查工具

## 特点

- Vue 3、Vite 5 - 极致快速
- Material You - 美观耐看
- pnpm、ESLint、Prettier - 开发便捷
- 完全静态作成

## 开发

请确保电脑上安装了 Node.js 与 pnpm。

```sh
git clone https://github.com/travellings-link/travellings-checktools.git
cd travellings-checktools
pnpm install
pnpm dev
pnpm lint
pnpm build
```
